# Sistema de Gestión de Alumnos

Sistema de gestión de alumnos desarrollado en Python como parte del examen parcial de Programación I.

## 📁 Estructura del Proyecto
gestion_alumnos.py # Programa completo (único archivo)
alumnos.json # Archivo de datos (se crea automáticamente)

## 🚀 Características

- Registrar alumnos con validaciones
- Listar todos los alumnos
- Buscar alumno por DNI
- Modificar datos de un alumno
- Eliminar alumno por DNI
- Estadísticas (total, promedio, mayor nota, aprobados, desaprobados)
- Persistencia en archivo JSON
- Reintentos en caso de error

## 🛠️ Requisitos

- Python 3.x
- No requiere librerías externas

## 📦 Instalación y Ejecución

1. Guardar la carpeta `Segundo-Parcial-Programacion-Lautaro-Rizzo`
2. Ejecutar el programa:
